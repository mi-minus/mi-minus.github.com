### 安装Redis
1. 下载 Redis-4.0.1
```
[下载链接](http://www.redis.cn/download.html)
```
2. 解压，编译
```sh
wget http://download.redis.io/releases/redis-4.0.1.tar.gz
tar xzf redis-4.0.1.tar.gz
cd redis-4.0.1
make
```
