### 查看ubuntu基本信息
0. 安装／卸载
```sh
sudo apt-get install xxx   : 安装方法1
dpkg -i xxx.deb            : 安装方法2
sudo apt-get purge xxx           : 卸载1
sudo apt-get remove --purge xxx  : 卸载2
sudo apt-get autoremove xxx      : 卸载3
```
