### ubuntu添加用户
1. 添加用户

    ```sh
    sudo useradd -m -s /bin/bash username
    ```
2. 更改密码
    ```sh
    sudo passwd username
    ```
3. 提升sudo权限
    ```sh
    sudo usermod -aG sudo username
    ```
    
    or 修改 /etc/sudoers文件，添加如下(参考里面自身语法)：
    
    ```sh
    username ALL=(ALL:ALL) ALL
    ```
